# Acronyms & Definitions

This document defines the acronyms used across the Kimberly repository and docs.

- ACID — Atomicity, Consistency, Isolation, Durability (database guarantees)
- ADR — Architecture Decision Record
- AI — Artificial Intelligence
- API — Application Programming Interface
- AR — Augmented Reality
- ASR — Automatic Speech Recognition
- CD — Continuous Delivery
- CI — Continuous Integration
- DAU — Daily Active Users
- DB — Database
- DELETE — HTTP DELETE method (remove resource)
- ELK — Elasticsearch / Logstash / Kibana (logging/observability stack)
- GDPR — General Data Protection Regulation (EU privacy law)
- GET — HTTP GET method (retrieve resource)
- GPL — GNU General Public License
- HTTPS — HTTP over TLS (secure HTTP)
- ISO — International Organization for Standardization
- JS — JavaScript
- JSON — JavaScript Object Notation
- JWT — JSON Web Token (authentication token format)
- KPI — Key Performance Indicator
- LLM — Large Language Model
- LR — Logistic Regression (or other abbreviation; context-specific)
- MAJOR — Semantic versioning: major release (breaking changes)
- MAU — Monthly Active Users
- MINOR — Semantic versioning: minor release (non-breaking feature)
- MIT — MIT License
- MVP — Minimum Viable Product
- NIST — National Institute of Standards and Technology
- NLP — Natural Language Processing
- OK — shorthand for success/acknowledgement (contextual)
- PATCH — Semantic versioning: patch release (bug fixes)
- PEP — Python Enhancement Proposal
- PM — Product Manager or Project Manager (context-specific)
- POST — HTTP POST method (create resource)
- PR — Pull Request
- PRD — Product Requirements Document
- QA — Quality Assurance
- README — repository high-level documentation file
- REST — Representational State Transfer (API style)
- SDK — Software Development Kit
- SLO — Service Level Objective
- SRE — Site Reliability Engineering
- TBD — To Be Determined
- TLS — Transport Layer Security (encryption protocol)
- TS — TypeScript
- TTL — Time To Live (cache or expiry duration)
- TTS — Text-to-Speech
- UI — User Interface
- UX — User Experience
- VM — Virtual Machine
- VR — Virtual Reality
- WCAG — Web Content Accessibility Guidelines
- YAML — YAML Ain't Markup Language (human-readable data serialization)

If you want other acronyms defined or to change definitions, add a PR against this file.
