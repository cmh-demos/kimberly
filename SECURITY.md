# Security

Vulnerability reporting, dependency scanning, and incident contacts.

## Vulnerability Reporting
- Report via GitHub issues or email: security@kimberly.ai.
- Response within 48 hours.

## Dependency Scanning
- Use tools like Snyk or Dependabot for automated scans.
- Review dependencies quarterly.

## Incident Response
- Contact: security@kimberly.ai.
- Follow NIST framework for incidents.
